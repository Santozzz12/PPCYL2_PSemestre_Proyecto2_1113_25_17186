# AcadNet - Sistema de Gestión Académica
Proyecto de Ingeniería desarrollado con arquitectura de microservicios (Flask + Django) implementando una Matriz Dispersa Ortogonal para la optimización de memoria.

### Para ejecutar localmente:
1. Backend: `python app.py` (Puerto 5000)
2. Frontend: `python manage.py runserver` (Puerto 8000)
